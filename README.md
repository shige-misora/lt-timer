# LT Timer - Lightning Talk Countdown

LT（ライトニングトーク）用のカウントダウンタイマーです。

## 機能

- **プリセット時間**: 1分、3分、5分、10分から選択
- **視覚的フィードバック**: 残り時間に応じた警告表示
  - 60% 以下: 黄色警告
  - 40% 以下: オレンジ警告
  - 20% 以下または残り30秒: 赤警告
  - 10% 以下または残り10秒: クリティカル警告（背景色変更）
- **プログレスバー**: 残り時間を視覚的に表示
- **一時停止/再開**: タイマーの一時停止と再開が可能
- **終了演出**: タイムアップ時に紙吹雪アニメーションと効果音
- **キーボードショートカット**:
  - `Space`: 一時停止/再開
  - `Escape`: リセット
- **レスポンシブデザイン**: モバイルデバイスにも対応
- **ダークモード対応**: システム設定に応じた表示

## 使い方

1. `index.html` をブラウザで開く
2. 希望する時間（1分、3分、5分、10分）を選択
3. 「スタート」ボタンをクリックしてカウントダウン開始
4. 必要に応じて一時停止やリセットが可能

## 技術仕様

- Pure HTML/CSS/JavaScript（外部ライブラリ不要）
- CSS変数によるテーマ管理
- Web Audio APIによる終了音
- CSS Animationsによるアニメーション効果
